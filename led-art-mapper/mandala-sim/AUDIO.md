# Demo audio (gitignored)

`demo.mp3` is not committed (it's a ~5MB binary and freely re-downloadable). To restore the "Play demo" button, drop any calm instrumental MP3 here named `demo.mp3`, or fetch the original:

Kevin MacLeod — "Deep Haze" (CC-BY, incompetech.com):
`curl -L -o demo.mp3 "https://incompetech.com/music/royalty-free/mp3-royaltyfree/Deep%20Haze.mp3"`

The simulator also works with the 🎤 Mic and 🎵 File buttons without this file.
