# Reference repos

Clone these here for reading — do not edit them.

```bash
cd reference-repos

# LED mapper math + path sampling reference
git clone https://github.com/jasoncoon/led-mapper

# Pixelblaze pattern language (JS-based per-pixel functions)
git clone https://github.com/simap/pixelblaze

# Pixelblaze Pixel Teleporter — 2D/3D LED visualiser protocol
git clone https://github.com/simap/PixelTeleporter

# Community Pixelblaze pattern library
git clone https://github.com/zranger1/PixelblazePatterns

# Browser-based WLED ledmap generator
git clone https://github.com/Aircoookie/WLED-Ledmap-Generator
```

These repos are reference only. All app code lives in `../app/`.
