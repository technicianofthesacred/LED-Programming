The Lit Mandala

Concentric ring strips sit behind a cut mandala. Each opening in the pattern is a window onto the ring behind it, so an effect running across the rings gets sculpted by the geometry into something far more intricate than the strips alone. Same imagined track throughout.

Ring effect

The trick that makes backlighting sing: the ring strips carry a smooth field (rise fast, fall slow), and the sharp detail comes entirely from the cut edges, not the animation. So you get intricate, crisp pattern and slow liquid motion at the same time, which is very hard to fake any other way. Layer two of these on alternate rings and the pattern reads as having depth.
