The Array

Sixteen separate strips placed in space. Instead of broadcasting the same signal to all of them, each mode below is a different logic for deciding where each part of the music goes. Same imagined late night track throughout. Watch how rarely they all light at once.

The same principle holds in every mode: a strip rises fast to a hit and falls slow, so motion reads as breath, not flicker. What changes is the address. Spreading the music across real space is what makes a piece feel like it has depth and intent rather than a single panel reacting. Run any two modes back to back and the difference in how the room feels is the whole argument.
