Lightweaver Studio · Design proposal

Wiring & Sizing, redesigned

The commissioning logic stays exactly as built — the state machines are good. This proposal replaces the presentation: one glanceable summary, one step at a time, questions in plain language, and power sizing finally in the same room as pixel sizing. Frames below are pixel mocks of the new panels; numbered chips map to the 20 changes in docs/wiring-sizing-ui-redesign.md.

02 · The bench check

One big question per screen

9·10
Guided wizard — the check takes over the panel. The old "visible from here" checkbox becomes the first screen with one honest button.
12
Recovery in place — every "Something's wrong" answer opens the fix right there (boundary slider, reverse direction, swap outputs), powered by the existing chase reducer.
11
Color quiz — nobody needs to know what "GRB" means. The strip lights up, you tap the color you see, twice. The order is derived.

03 · Size & Power

Pixels and amps in the same room

15·16
Merged panel — strip sizing and the power budget (today stranded in the unreachable legacy Devices panel) become one view. Draw, supply, and headroom as tiles with a meter.
17
Density you can see — the 30/60/144 picker renders real dot spacing.
18
Drag to size — pull the strip's end handle on the artwork; the number input stays for fine adjust. Length shown in cm and LEDs.
19
Advice, not just numbers — over budget produces a suggestion derived from the math, not a red label.
